# 雨の終わりに ― 私立探偵・灰崎の事件簿 ―

約10分で遊べる、昭和レトロ風の推理アドベンチャーゲーム。
逆転裁判・神宮寺三郎・ファミコン探偵倶楽部へのオマージュ作品です。

雨の夜、ベストセラー推理作家が書斎で毒死。机には遺書。
自殺で片づきそうな事件に、私立探偵・灰崎修が「待った」をかける――。

## 遊び方

**`ame_no_owarini.html` をダブルクリックするだけ。**
ブラウザでそのまま動きます（インストール・通信・サーバー不要の単一HTMLファイル）。

- クリック または Enter でテキストを読み進める
- 画面右上の「音：オン／オフ」でBGM・効果音を切り替え
- プレイ時間：約10分

## ゲームの流れ

| パート | スタイル | 内容 |
|---|---|---|
| 調査 | 神宮寺三郎／ファミ探風 | 「調べる・話す・移動する・考える」のコマンド選択で現場検証と聞き込み。証拠は探偵手帳に記録される |
| 中間推理 | ファミ探風 | 警部に「自殺と食い違う証拠」を突きつける推理クイズ |
| 対決 | 逆転裁判風 | 容疑者の証言を「ゆさぶる」「証拠をつきつける」で矛盾を突く尋問。お手つきはライフ（◆×5）が減り、ゼロで仕切り直し |

## 技術メモ

- 依存ライブラリなし（素のHTML/CSS/JavaScript 1ファイル）
- 背景・人物バストアップ・証拠品アイコンはすべてインラインSVG（雨はSMILアニメーション）
- BGM・雨音・効果音は Web Audio API によるリアルタイム生成（音源ファイルなし）
- `game_server.js` は開発時のプレビュー用簡易サーバー（Node.js）。**ゲームの動作には不要**

## デバッグ用URLパラメータ（開発者向け）

- `?dbg=quiz` … 中間推理クイズから開始
- `?dbg=trial` … 終盤の尋問パートから開始
